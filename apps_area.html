What is LiFi?


LiFi is a developing technology, in which any type of data is transmitted using light rays. It is done by encoding the data (ie. to be transferred) into binary form and blinking the light source (transmitter) according to it in a very certain time interval and when that light rays fall on a light sensative sensor it is converted into binary form and is decoded by the receiver device.
